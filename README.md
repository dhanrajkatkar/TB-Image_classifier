# TB-Image_classifier

This is a tuberculosis detection task repository created for Longevity Scientific Resort Computer vision assignment task

## Dataset

The dataset used is available [here](https://www.kaggle.com/tawsifurrahman/tuberculosis-tb-chest-xray-dataset).

## Installing requirements
Install the dependencies and devDependencies and start the server.

```sh
pip install -r requirements.txt
```
